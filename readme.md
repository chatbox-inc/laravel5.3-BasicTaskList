# Laravel5.3 Basic Task List

5.2 のドキュメントを参考に 5.3 で Basic Task List をつくりました。

1. ディレクトリをクローン

```
git clone https://github.com/sshono1210/oic-ec.git
```

2. composerをインストール

```
composer install
```

3. .env.exampleをコピーして.envファイルを作成

```
cp .env.example .env
```

4. キーの作成

```
php artisan key:generate
```
